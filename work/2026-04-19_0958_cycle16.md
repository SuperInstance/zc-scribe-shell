## Step 2: Scoring Rubric (0-10)

*   **0-2 (Poor):** Missing README, or README is a placeholder. No context, no usage instructions.
*   **3-4 (Basic):** README exists with a brief description. May have broken links or incomplete setup steps.
*   **5-6 (Adequate):** Clear purpose, basic installation/usage. May lack examples, architecture diagrams, or contribution guidelines.
*   **7-8 (Good):** Comprehensive README with purpose, setup, usage examples, and API overview. Links to further documentation.
*   **9-10 (Excellent):** All "Good" criteria, plus clear architecture diagrams, contribution guidelines, troubleshooting, and links to related projects or research context.
